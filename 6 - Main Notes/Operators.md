# Operators

Los operadores son símbolos especiales que nos permiten hacer operaciones especificas y regresar un resultado

| Operators            | Precedence                                                      |
| -------------------- | --------------------------------------------------------------- |
| postfix              | `expr++` `expr--`                                               |
| unary                | `++expr` `--expr` `+expr` `-expr` `~` `!`                       |
| multiplicative       | `*` `/` `%`                                                     |
| additive             | `+` `-`                                                         |
| shift                | `<<` `>>` `>>>`                                                 |
| relational           | `<` `>` `<=` `>=` `instanceof`                                  |
| equality             | `==` `!=`                                                       |
| bitwise AND          | `&`                                                             |
| bitwise exclusive OR | `^`                                                             |
| bitwise inclusive OR | `\|`                                                            |
| logical AND          | `&&`                                                            |
| logical OR           | `\|`                                                            |
| ternary              | `? :`                                                           |
| assignment           | `=` `+=` `-=` `*=` `/=` `%=` `&=` `^=` `\|=` `<<=` `>>=` `>>>=` |
